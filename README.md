# World Brief

## 概要

World Brief は、英語の主要報道機関の記事をもとに、日本語で毎朝配信する世界情勢ブリーフです。

目的は、5分程度で世界情勢の重要な動きを把握できることです。

---

## 特徴

- 英語ニュースのみを情報源とする
- 日本語で簡潔に要約
- 「なぜ重要か」を重視
- 情報源へのリンクを掲載
- HTMLとして保存・公開
- GitHub Pagesで閲覧可能
